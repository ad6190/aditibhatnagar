# Feature Spec: Add a Footer with Copyright Year

**Issue:** #3
**Feature:** Footer with copyright year
**Status:** Spec Complete
**Date:** 2026-05-25

---

## One-Liner

Add a `<footer>` element to `Layout.tsx` that displays `© <current-year> aditi.` on every page.

---

## Scope

**In scope:**
- Footer in `Layout.tsx` (site-wide)
- Dynamic copyright year via `new Date().getFullYear()`
- Responsive layout using Tailwind utility classes
- Secondary tagline ("obsidian + react")
- Semantic HTML (`<footer>` element)

**Out of scope:**
- Social links
- Navigation links in footer
- Dark mode toggle
- Analytics

---

---

## Component Design

**File to modify:** `blog-site/site/src/components/Layout.tsx`

**New footer structure:**
```tsx
<footer className="border-t py-8 px-6 lg:px-8">
  <div className="mx-auto max-w-[1400px] flex justify-between items-center text-sm text-muted-foreground">
    <span>&copy; {new Date().getFullYear()} aditi.</span>
    <span className="text-xs">obsidian + react</span>
  </div>
</footer>
```

**Classes used:**
- `border-t` — top border separating footer from content
- `py-8 px-6 lg:px-8` — consistent padding matching the nav
- `mx-auto max-w-[1400px]` — consistent max-width matching the nav
- `flex justify-between items-center` — two-column layout
- `text-sm text-muted-foreground` — subtle text color
- `text-xs` — slightly smaller for the tagline

- Confirm footer appears at the bottom of each page

---

## Notes

- The `Layout` component already uses `<Outlet />` from React Router, so the footer will automatically appear on all routes without any per-page changes.
- `new Date().getFullYear()` is evaluated at render time, so it will always show the current year.
- Using `text-muted-foreground` (a CSS variable from the design system) ensures the footer respects future theme changes.
